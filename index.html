Why Put Up With Weak Security?

WiFi Password Sharing Is Risky

The wifi password could be stolen from old devices when they're thrown away or sold. It is tedious and time consuming to redo the setup on all your devices. Malicious or breached devices can also snoop or intercept traffic.

Insecure IoT Devices Cause Breaches

Many IoT devices are insecure, rarely updated with security fixes, and may not respect your home privacy when they connect to the cloud.

Limited Network Isolation Compounds Risk

Why should your vacuum be able talk to your doorbell? Inadequate network isolation makes breaches worse

Your Network Leaks Data To Cloud Companies

Keep your home network private. Block constant data collection from IOT devices, websites, and mobile phones.

Get started with SPR

We offer hardware for running SPR and it can run on a wide variety of Linux systems that support Docker. Raspberry PI4/5 and Compute Modules are widely tested.

Check out the setup guide. SPR is open source and can run in Docker containers as a cloud-only VPN or as a WiFi router.

Prebuilt containers are available for ARM64 and AMD64, OS Images are available for Raspberry Pi and ClearFog.

Run as a Host Router & Firewall with Docker 

git clone https://github.com/spr-networks/super
cd super
bash base/setup.sh
docker compose up --pull always -d

This will manage the system network, firewall, and interfaces. Note: this replaces network manager, see the setup guide for details

VPN Only Quick Setup 

sudo bash -c "$(curl -fsSL https://raw.github.com/spr-networks/super/master/virtual_install.sh)"

The script installs and & starts SPR VPN in a virtual network. See the virtual setup guide for more info
