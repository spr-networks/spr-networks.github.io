Starter Kit

Ready to try SPR? You can run the software on a Raspberry Pi 4. Grab the Complete Starter Kit for everything you need to get started.

Features

  • Device Isolation with AP/VLAN
  • Per-device passwords with WPA2, WPA3
  • Groups & Tags
  • DNS Ad Blocking & Per-Device DNS rules
  • Wireguard VPN
  • Powerful homelab additions with 8GB RAM & 64GB USB SSD

Wireless Capabilities

  • WiFi 5 with 80 Mhz channels
  • Top speeds ranging to 600mbps
  • 5Ghz or 2.4Ghz band
  • Built-in PI WLAN can be used as a wifi client for sharing wireless uplink
  • Mesh supported with SPR PLUS
Note: Only one WiFi channel supported per WiFi Adapter. Multiple Adapters can be run off of the Raspberry Pi with a powered USB Hub
About the Custom Cases

  • Built locally in Washington State
  • Hard Plastic, removable case without exterior screws
  • Supports NetGear A6210 WiFi Dongle
  • Male to Female USB3 Adapter included
  • Great as a travel router

Clearfog C9130 Pro Dev Kits

Coming soon! This Pro Kit is designed to provide high performance and flexibility. It features two upgradable mini-PCIe slots and an M.2 slot, allowing for expansion and customization. Additionally, it is equipped with a 10GbE SFP port which can handle high-speed data transfer. The device also supports WiFi 6-e cards, providing the latest wireless technology with extended frequencies. This device can readily run multiple high-speed WiFi cards with multiple APs.

Board Info
  • 1 x SFP+ 10 GbE port for high-speed data transfer
  • 1 x Port dedicated Ethernet
  • 5 x Switched Ethernet RJ45 10/100/1000
  • 2 x upgradable MiniPCIe slots for expansion and customization
  • 1 x M.2 slot for additional storage
  • Marvell OCTEON based CN9130 Quad core Arm Cortex A72 up to 2.2 GHz
  • 4GB of RAM
  • 8GB eMMC storage
  • microSD slot for additoinal storage
  • Micro USB Debugging and Programming
  • SIM holder for cellular connectivity
  • 1x USB 3.0 for high-speed data transfer
