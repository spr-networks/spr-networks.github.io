SPR on the SolidRun Clearfog

We offer a complete SolidRun kit for running SPR as a primary router or mesh node. These devices carry a 10Gbps SFP port and a 5xport 1gbps switch in addition to a dedicated 1gbps uplink ethernet port.

Clearfog C9130 Pro Dev Kits

Board Info
  • Packaged with MT7915e 3x3 DBDC WiFi 6-e card and External Antennas
  • 1 x SFP+ 10 GbE port for high-speed data transfer
  • 1 x Port dedicated Ethernet
  • 5 x Switched Ethernet RJ45 10/100/1000
  • 2 x upgradable MiniPCIe slots for expansion and customization, one of which hosts WiFi
  • 1 x M.2 slot for additional storage
  • Marvell OCTEON based CN9130 Quad core Arm Cortex A72 up to 2.2 GHz
  • 4GB of RAM
  • 8GB eMMC storage
  • microSD slot for additoinal storage
  • Micro USB Debugging and Programming
  • SIM holder for cellular connectivity
  • 1x USB 3.0 for high-speed data transfer
